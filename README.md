# biocad-json-parsing
Программа, которая считывает данные из файла data.json (src/main/resources/data.json) и выводит их в виде таблицы (на экран или в консоль)

## Запуск и компиляция
###  Компиляция
```
mvn clean install assembly:single
```

### Запуск (Вар. 1)
```
java -jar biocad-json-parser.jar
```
